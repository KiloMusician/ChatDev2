# Challenge: Keyboard Ninja (VS Code)

**Objective:** complete a mini workflow using only keyboard shortcuts.

## Tasks
1. Open `START_HERE.md` via Quick Open (`Ctrl/Cmd+P`).
2. Open the Command Palette (`Ctrl/Cmd+Shift+P`) and run:
   - `View: Toggle Terminal`
3. Search across the workspace (`Ctrl/Cmd+Shift+F`) for: `DevMentor`
4. Return to Explorer (`Ctrl/Cmd+Shift+E`)
5. Create a new file in `.devmentor/notes/` named `keyboard_ninja.md`

Write 3 lines:
- Which shortcut felt most useful?
- Which shortcut do you want to memorize next?
- What slowed you down?

## Reward suggestion (manual for v0)
Add `Keyboard Ninja` to `.devmentor/state.json` under `achievements`.
