# AI Tools in VS Code 01 — AI Workflow (Use + Verify)

**Goal:** learn the correct loop: AI suggestion → apply → verify.

## Setup
If you have Copilot installed, open Copilot Chat.
If you don't, skip and focus on the workflow steps.

## Exercise
Pick a small file and ask an AI tool:
> “Suggest a refactor that improves readability without changing behavior.”

Then:
1. Apply the refactor
2. **Verify**:
   - run any tests if present
   - run lint/format if configured
   - manually run the script if needed
3. Keep only what you can justify

## Rule
AI output is a draft. **Verification is the lesson.**
