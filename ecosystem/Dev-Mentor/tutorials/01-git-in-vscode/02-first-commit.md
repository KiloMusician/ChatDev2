# Git in VS Code 02 — Your First Intentional Commit

## Do this now
1. In SCM view, stage a single file (not everything).
2. Write a commit message:
   - Prefer: `chore: setup devmentor runtime`
3. Commit.
4. Optional: open GitLens and inspect the commit.

## Reflection (save-game note)
Create this file (gitignored):
`.devmentor/notes/first_commit.md`

Write 2–5 sentences:
- What did you stage?
- Why didn’t you stage everything?
- What would you do differently next time?
