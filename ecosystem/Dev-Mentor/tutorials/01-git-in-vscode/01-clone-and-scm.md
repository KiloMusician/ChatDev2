# Git in VS Code 01 — Clone + Source Control View

**Goal:** use Git from inside VS Code, not just the terminal.

## Do this now
1. Open Source Control view (SCM icon).
2. If this repo isn't a git repo yet, initialize it:
   - Command Palette → `Git: Initialize Repository`
3. Make a small change anywhere (e.g., add a line to `.gitignore`).
4. Observe the file changes in the SCM view.

## What to notice
- VS Code stages and commits are *actions* you can learn, not magic.
- Staging is the “shopping cart,” commit is the “receipt.”
