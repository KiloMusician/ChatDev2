# VS Code Basics 02 — Keyboard Shortcuts (No-Mouse Warmup)

**Goal:** build muscle memory for core navigation.

## Do this now
Try these without touching the mouse:
- Open Command Palette: `Ctrl/Cmd+Shift+P`
- Quick Open: `Ctrl/Cmd+P`
- Toggle Terminal: ``Ctrl/Cmd+` ``
- Search in file: `Ctrl/Cmd+F`
- Global search: `Ctrl/Cmd+Shift+F`
- Toggle Sidebar: `Ctrl/Cmd+B`
