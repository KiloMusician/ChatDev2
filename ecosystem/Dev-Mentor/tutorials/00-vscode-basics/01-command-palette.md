# VS Code Basics 01 — Command Palette Mastery

**Goal:** learn to drive VS Code through the Command Palette.

## Do this now
1. Open Command Palette:
   - Windows/Linux: `Ctrl+Shift+P`
   - macOS: `Cmd+Shift+P`
2. Type: `View: Toggle Terminal` → run it.
3. Type: `Preferences: Open Settings (JSON)` → open it.
4. Return to the workspace Settings UI (optional): `Preferences: Open Settings`
