# DevMentor (VS Code-Native)

DevMentor is a **VS Code-native mentorship repository**. The repository is the product.

## The Trigger
Open this folder in VS Code:

```bash
code .
```

That’s the “boot.” DevMentor runs through:
- **VS Code Tasks**
- **Workspace configuration**
- **Tutorials and challenges**
- **Local scripts (portable, offline-friendly)**

## Start
1. Open **START_HERE.md**
2. Run the task: **DevMentor: Start/Resume**
   - Command Palette → `Tasks: Run Task` → select it

## Portability (Replit → ZIP → VS Code)
At any time:
- Run task: **DevMentor: Export Portable ZIP**
- Unzip anywhere
- Open in VS Code
- Run task: **DevMentor: Import Portable ZIP**
- Continue seamlessly

## What’s included in this template (v0)
- Progressive `.vscode/` config
- Boot + resume scripts
- Portable state export/import
- First-hour tutorials (VS Code basics, Git in VS Code, AI workflow basics)
- Two starter challenges + validator
