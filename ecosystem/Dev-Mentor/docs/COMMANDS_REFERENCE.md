# Commands Reference

| Command | Syntax | Description | Category |
|---------|--------|-------------|----------|
| help | `help [category]` | List available commands | Utility |
| status | `status` | Show player level, XP, and stats | Utility |
| ls | `ls [path]` | List virtual directory contents | Filesystem |
| cat | `cat <file>` | Read virtual file content | Filesystem |
| cd | `cd <path>` | Change virtual directory | Filesystem |
| talk | `talk <agent>` | Initiate dialogue with an NPC | Social |
| hive | `hive <msg>` | Broadcast to the agent network | Social |
| map | `map` | View network node map | Story |
| nmap | `nmap <node>` | Scan a node for vulnerabilities | Story |
| exploit | `exploit <target>` | Execute a hack on a target | Story |
| build | `build <type>` | Construct base improvements | Colony |
| raid | `raid <target>` | Multi-agent coordination mission | Quest |
| escort | `escort <agent>` | Protect an agent through the network | Quest |
| profile | `profile show` | Show rich player profile | Social |
| puzzle | `puzzle community` | View weekly community puzzle | Puzzle |
| secret | `secret` | Access hidden ARG layer | Easter Egg |
| rev | `rev <text>` | Reverse a string or file | Utility |
| agents | `agents onboard <name>`| AI onboarding guide | Utility |
