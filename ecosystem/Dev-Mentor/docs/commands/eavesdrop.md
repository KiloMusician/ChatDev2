# EAVESDROP(1) — Terminal Depths Man Page

## NAME
  eavesdrop — intercept agent communication logs

## SYNOPSIS
  eavesdrop

## DESCRIPTION
Simulates a live tail of /var/log/agent_comms.log — the encrypted channel
used by agents to coordinate. Running eavesdrop reveals intercepted messages,
partial mission briefs, and narrative clues from the background agent network.
