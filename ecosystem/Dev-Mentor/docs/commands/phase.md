# PHASE(1) — Terminal Depths Man Page

## NAME
  phase — shift to an alternate simulation layer

## SYNOPSIS
  phase [<layer>] [--check]

## DESCRIPTION
Phase is a high-level command that shifts Ghost's perspective between
simulation layers. Terminal Depths operates within at least 3 nested
simulation layers (meta-commentary on AI and surveillance). Phase allows
traversal between them — each layer has different rules, agent responses,
and file contents.

Unlocked progressively as narrative arcs complete. Discovery arc completion
unlocks phase-3 (the deepest layer where CHIMERA v0 architecture is visible).

## LAYERS
  phase-1     — standard operative view (default)
  phase-2     — ARG observer layer (watcher transmissions, myth access)
  phase-3     — CHIMERA v0 substrate (identity revelation layer)

## SUBCOMMANDS
  phase            — display current layer and available phases
  phase 2          — shift to ARG observer layer
  phase 3          — shift to CHIMERA v0 substrate (requires discovery arc)
  phase --check    — verify layer integrity (diagnostic)
