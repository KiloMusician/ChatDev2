# FEED(1) — Terminal Depths Man Page

## NAME
  feed — feed information (true or false) to an agent

## SYNOPSIS
  feed <agent> [disinfo|truth] [<message>]

## DESCRIPTION
Use your position to feed targeted intelligence to an agent. With `disinfo`,
plant false data to misdirect an enemy. With `truth`, share real intel to
build trust faster (trust +5 instead of +2).

In the mole arc, `feed <mole_agent> disinfo` is one of the three resolution
paths. If the mole passes your disinfo to NexusCorp, you can trace the relay
and get irrefutable confirmation.

## MODES
  feed <agent>              — neutral intel share (minor trust boost)
  feed <agent> truth        — share verified intel (+5 trust)
  feed <agent> disinfo      — plant false data (risk/reward play)
  feed <agent> disinfo <msg>— custom disinformation message

## EXAMPLES
  feed ada truth
  feed spartacus disinfo
  feed cypher disinfo "CHIMERA kill switch is at /opt/chimera/keys/kill.key"
