# BAZAAR(1) — Terminal Depths Man Page

## NAME
  bazaar — The Simulation's market node — buy and sell items

## SYNOPSIS
  bazaar [list [<category>]|buy <item-id>|sell <item-id>|status]

## DESCRIPTION
The Bazaar is the in-game legitimate market. Browse items by category, purchase
gear with credits, or liquidate inventory. Items range from scripts and tools
to narrative artefacts and augment components.

## SUBCOMMANDS
  list [cat]    Browse items, optionally filtered by category
  buy <id>      Purchase item by ID
  sell <id>     Sell an item from inventory
  status        Show your active listings and wallet

## EXAMPLES
  bazaar list
  bazaar list tools
  bazaar buy phantom-blade
  bazaar sell old-cipher
