# ART

  **ASCII art gallery. Cycles through cyberpunk ASCII art.**

  ## Usage
  ```
  art [subcommand] [args]
  ```

  ## Description
  ASCII art gallery. Cycles through cyberpunk ASCII art. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `art` — default view
  - `art help` — show detailed assistance

  ## Examples
  ```
  art next
  ```

  ## Notes
  In a world of data, beauty is the only anomaly.
