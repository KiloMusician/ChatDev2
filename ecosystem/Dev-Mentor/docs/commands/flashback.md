# FLASHBACK

  **ZERO's historical arc. Shows redacted archive entries from /opt/special_circumstances/ and /home/ghost/.zero/ files.**

  ## Usage
  ```
  flashback [subcommand] [args]
  ```

  ## Description
  ZERO's historical arc. Shows redacted archive entries from /opt/special_circumstances/ and /home/ghost/.zero/ files. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `flashback` — default view
  - `flashback help` — show detailed assistance

  ## Examples
  ```
  flashback view origin
  ```

  ## Notes
  Memory is the only thing the reset cannot touch.
