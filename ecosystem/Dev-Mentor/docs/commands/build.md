# build

Construct upgrades in SimulatedVerse.

## Usage
`build [list|construct <type>|status|upgrade <type>]`

## Description
Construct and manage buildings in the SimulatedVerse to gain bonuses.
Types:
- `lab`: +15% research XP (200 credits)
- `comms_hub`: +20% hive message rate (150 credits)
- `safehouse`: -10% trace accumulation (300 credits)
- `relay`: +3 nodes_capacity (100 credits)
- `armory`: +20% exploit damage in boss fights (250 credits)

## Building Status
`build status` shows completion progress for ongoing constructions.
Build time is approximately 60 seconds.

## Rewards
- Passive bonuses per building type.
