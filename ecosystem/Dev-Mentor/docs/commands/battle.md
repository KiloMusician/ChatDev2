# BATTLE(1) — Terminal Depths Man Page

## NAME
  battle — engage in agent combat

## SYNOPSIS
  battle <agent> [mode]

## DESCRIPTION
Initiates a turn-based combat encounter with an agent or node. Combat uses
your skill stats (hack skill for cyber attacks, stealth for evasion) and
available tools. Winning awards XP, credits, and may unlock story content.

## EXAMPLES

  battle chimera-guardian stealth
  battle rogue-agent
