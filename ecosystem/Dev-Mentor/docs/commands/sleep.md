# SLEEP(1) — Terminal Depths Man Page

## NAME
  sleep — pause execution for a duration

## SYNOPSIS
  sleep <seconds>

## DESCRIPTION
Pauses the terminal for a specified number of seconds. Useful in scripts
between operations. Unlike meditate, sleep does not award XP. Maximum 60s
in interactive mode.

## EXAMPLES
  sleep 5
  sleep 30
