# BLACKMAIL(1) — Terminal Depths Man Page

## NAME
  blackmail — coerce an agent into becoming a double operative

## SYNOPSIS
  blackmail <agent> [--leverage <evidence_file>]

## DESCRIPTION
One of the three mole resolution paths. Instead of exposing the mole
publicly, use gathered evidence to coerce them into working for both sides.
A double agent provides insider intel on NexusCorp operations but remains
a calculated risk — they can turn again if trust falls below 20.

Requires: `arc_mole_identified` beat, at least 2 clue files in evidence.

## CONSEQUENCES
  Success:
    • Mole becomes double agent (visible in `agents` list with ⚡ marker)
    • Passive NexusCorp intel every 3 sessions
    • Trust locked at 15 (uneasy alliance)
    • XP +150, achievement: DOUBLE_AGENT

  Failure (insufficient evidence):
    • Mole becomes hostile, trust → 0
    • NexusCorp threat level +2

## EXAMPLES
  blackmail spartacus
  blackmail ada --leverage /home/ghost/.mole_trace
