# rivalry

## NAME
rivalry — manage agent tension and interpersonal conflict

## SYNOPSIS
```
rivalry status
rivalry escalate <agent1> <agent2>
rivalry mediate <agent1> <agent2>
```

## DESCRIPTION
Some agents have natural tension axes. Escalating rivalry generates dramatic hive events and story beats. Mediating costs social XP but improves both relationships.

## TENSION PAIRS
- ada ↔ cypher (ethics vs pragmatism)
- gordon ↔ raven (strategy vs instinct)
- nova ↔ serena (corporate vs convergence)
- malice ↔ zero (chaos vs order)

## SUBCOMMANDS
- `rivalry status` — show all tension bars (0-100)
- `rivalry escalate <a> <b>` — raise tension +20, costs 10 XP
- `rivalry mediate <a> <b>` — lower tension -15, costs 20 XP
