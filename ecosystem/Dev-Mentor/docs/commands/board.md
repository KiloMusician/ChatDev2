# BOARD

  **Faction board game overlay. Territory control on the network map.**

  ## Usage
  ```
  board [subcommand] [args]
  ```

  ## Description
  Faction board game overlay. Territory control on the network map. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `board` — default view
  - `board help` — show detailed assistance

  ## Examples
  ```
  board status
  ```

  ## Notes
  The network is a board. The agents are pieces. You are the player.
