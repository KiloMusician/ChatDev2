# endless

## NAME
endless — post-story procedural content mode

## SYNOPSIS
```
endless start
endless next
endless status
```

## DESCRIPTION
After completing the main story arc, endless mode provides infinite procedural content with escalating difficulty. Each wave generates a new procgen quest and a harder node to infiltrate.

## UNLOCK
Requires `mission_complete` achievement. See: `ascend-story`

## SUBCOMMANDS
- `endless start` — initiate endless mode
- `endless next` — generate next wave (harder each time)
- `endless status` — view current wave number and cumulative rewards
