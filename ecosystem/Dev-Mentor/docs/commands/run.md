# RUN(1) — Terminal Depths Man Page

## NAME
  run — execute a script or binary

## SYNOPSIS
  run <name> [args...]

## DESCRIPTION
Short alias for script run. Executes a named script from the script library
with optional arguments.

## EXAMPLES
  run recon.js --target nexus-core
  run exfil.js /etc/shadow
  run loop.js
