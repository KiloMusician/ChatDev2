# IFCONFIG(1) — Terminal Depths Man Page

## NAME
  ifconfig — display virtual network interface configuration

## SYNOPSIS

## DESCRIPTION
Shows the Terminal Depths virtual network topology: virtual NICs, IP
assignments, MAC addresses, and active connections. Useful for target
reconnaissance before running nmap or hack.
