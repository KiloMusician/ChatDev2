# DICE

  **Roll dice (default 1d6). Luck attribute modifies rolls.**

  ## Usage
  ```
  dice [subcommand] [args]
  ```

  ## Description
  Roll dice (default 1d6). Luck attribute modifies rolls. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `dice` — default view
  - `dice help` — show detailed assistance

  ## Examples
  ```
  dice 20 2
  ```

  ## Notes
  Luck is not just a stat; it's a ghost in the machine.
