# AUGMENT(1) — Terminal Depths Man Page

## NAME
  augment — install a cybernetic enhancement module

## SYNOPSIS
  augment <module> [--slot <slot>]

## DESCRIPTION
Augmentations are persistent upgrades to Ghost's capabilities. Each augment
occupies a slot (neural, optical, physical, digital). Augments provide passive
bonuses to skill checks, unlock new command variants, or gate story content.

Augments are purchased with CRED (in-game currency earned through missions).

## AUGMENT SLOTS
  neural     — intelligence, AI interaction, memory capacity
  optical    — recon, steganography detection, visual analysis
  physical   — (cosmetic for a digital entity, but affects avatar)
  digital    — hacking success rates, encryption cracking, speed

## AVAILABLE AUGMENTS
  neural-mk1    — +10% AI skill XP gain
  optical-tap   — reveals hidden files in ls output (ls -la auto)
  ghost-wire    — +15% networking XP, reveals node connections on map
  cipherskin    — +20 flat bonus to crack difficulty checks
  void-eye      — detects mole transmission patterns passively

## EXAMPLES
  augment neural-mk1
  augment optical-tap --slot optical
