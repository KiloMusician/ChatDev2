# turret

Place defensive scripts on nodes.

## Usage
`turret [place <node>|list|remove <node>|upgrade <node>]`

## Description
Place defensive scripts on captured nodes to reduce trace accumulation.
Each turret costs 100 credits. Max 3 per node.

## Stats
- Standard: -5% trace accumulation per tick.
- Upgraded: -10% trace accumulation per tick.

## Requirements
- Node must be colonized.

## Rewards
- Reduced trace risk.
