# WARP(1) — Terminal Depths Man Page

## NAME
  warp — instantly travel to a discovered network node

## SYNOPSIS
  warp <node>

## DESCRIPTION
Warp provides instant transit to any previously discovered node without
requiring a new hack session. You must have at least read access to the
target node (prior successful connection or scout). Warp bypasses SSH
handshaking but does not bypass file permission checks.

Useful for quickly returning to nodes during multi-step arcs.

## REQUIREMENTS
  Node must be in your discovered node list (check: map)
  You must have had at least one successful connection
  Some nodes lock warp after hostile detection (use scan to re-check)

## EXAMPLES
  warp nexus-gateway
  warp chimera-control
  warp darknet-relay-7
  warp /home/ghost             (return home)

## NODE SHORTCUTS
  warp home        — return to /home/ghost (always available)
  warp last        — warp to previously visited node
  warp safe        — warp to nearest safe node (resistance cell)
