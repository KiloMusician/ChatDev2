# TALK(1) — Terminal Depths Man Page

## NAME
  talk — initiate dialogue with an unlocked agent

## SYNOPSIS
  talk <agent> [topic]

## DESCRIPTION
Opens a dialogue session with a network contact. Talking builds trust over
time. Some agents gate critical intel behind trust thresholds; others reveal
hidden agendas only when fully trusted. The `topic` parameter lets you steer
conversation toward specific subjects.

## TOPICS
  trust       — directly ask the agent about your relationship
  mission     — discuss current mission objectives
  chimera     — probe what the agent knows about CHIMERA
  mole        — ask about the traitor in the network (late-game)
  watcher     — ask about the mysterious WATCHER entity

## EXAMPLES
  talk ada
  talk cypher mission
  talk ada trust
  talk raven mole

## TRUST EFFECTS
Each successful talk increases trust by +2. At trust ≥ 25, dossier unlocks.
At trust ≥ 50, deeper mission intel. At trust ≥ 80, hidden agenda revealed.
