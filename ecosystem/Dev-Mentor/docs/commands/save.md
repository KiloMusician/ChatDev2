# SAVE(1) — Terminal Depths Man Page

## NAME
  save — manually save game state

## SYNOPSIS
  save

## DESCRIPTION
Triggers a manual save of the current game state to the session file. The
game auto-saves after every command, but save lets you create an explicit
checkpoint. Save data includes XP, story beats, agent relationships, and VFS.

## EXAMPLES
  save
