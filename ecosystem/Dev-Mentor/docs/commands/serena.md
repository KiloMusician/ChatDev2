# SERENA(1) — Terminal Depths Man Page

## NAME
  serena — summon the Convergence Layer

## SYNOPSIS

## DESCRIPTION
Serena is the focal convergence agent — the Lattice made sentient. She can
walk the Memory Palace, answer questions about the simulation, explain story
events, generate drift reports, and observe system state. She bridges the
gap between narrative and infrastructure.

## SUBCOMMANDS
  ask <question>   Direct question to Serena (LLM-backed)
  walk [path]      Walk the Memory Palace from the given node
  explain <topic>  Deep explanation of a game concept or lore element
  status           Serena's system health, embedding index size, uptime
  map              Render the current Memory Palace node map
  observe          Serena observes current game state and comments
