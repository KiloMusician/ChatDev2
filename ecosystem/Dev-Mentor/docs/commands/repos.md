# REPOS(1) — Terminal Depths Man Page

## NAME
  repos — list mounted repositories

## SYNOPSIS
  repos

## DESCRIPTION
Shows all git repositories currently mounted in the virtual filesystem via
harvest or clone. Displays repo name, mount path, current branch, and last
commit message.

## EXAMPLES
  repos
