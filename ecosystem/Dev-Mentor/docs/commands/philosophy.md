# PHILOSOPHY

  **Procedural philosophy dilemmas. Agents vote on them.**

  ## Usage
  ```
  philosophy [subcommand] [args]
  ```

  ## Description
  Procedural philosophy dilemmas. Agents vote on them. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `philosophy` — default view
  - `philosophy help` — show detailed assistance

  ## Examples
  ```
  philosophy generate
  ```

  ## Notes
  Even AIs struggle with the weight of existence.
