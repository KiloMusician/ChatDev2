# CERTIFY(1) — Terminal Depths Man Page

## NAME
  certify — in-game certificate management (FutureLearn/Coursera-inspired)

## SYNOPSIS
  certify [list|check <id>|claim <id>]

## DESCRIPTION
Earn certificates for completing learning paths, achieving milestones, and
mastering skill areas. Modelled on FutureLearn (90% completion required) and
Coursera (specialisation certificates). Certificates appear in your portfolio.

## CERTIFICATES

  soc-l1          SOC Analyst L1 Certificate
  jr-pentester    Jr Penetration Tester Certificate
  red-team        Red Team Operator Certificate
  attck-scholar   ATT&CK Scholar Certificate
  arch-master     Architecture Master Certificate

## SUBCOMMANDS
  list            Show all certificates and earned status
  check <id>      Check requirements for a certificate
  claim <id>      Claim a certificate if requirements are met
