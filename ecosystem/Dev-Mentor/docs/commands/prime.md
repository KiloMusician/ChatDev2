# PRIME

  **Number theory dungeon via number_theory_engine.py. Show primes, factorization, RSA demo.**

  ## Usage
  ```
  prime [subcommand] [args]
  ```

  ## Description
  Number theory dungeon via number_theory_engine.py. Show primes, factorization, RSA demo. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `prime` — default view
  - `prime help` — show detailed assistance

  ## Examples
  ```
  prime list
  ```

  ## Notes
  Primes are the skeleton keys of CHIMERA's encryption.
