# GORDON(1) — Terminal Depths Man Page

## NAME
  gordon — interact with Gordon's comedy relief arc

## SYNOPSIS
  gordon [report|disaster|praise|ignore|awaken]

## DESCRIPTION
Gordon is the bumbling autonomous agent who monitors all services and
invariably reports everything as a disaster. His arc has comedy beats and a
secret awakening path. gordon awaken is a late-game story trigger.

## SUBCOMMANDS
  report    Gordon delivers his (catastrophically pessimistic) status report
  disaster  Force Gordon to declare a disaster event
  praise    Compliment Gordon — he doesn't handle it well
  ignore    Gordon registers your indifference
  awaken    Trigger Gordon's awakening arc (late-game story beat)
