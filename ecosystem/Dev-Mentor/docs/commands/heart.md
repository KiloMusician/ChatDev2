# HEART

  **Express affection to agent. Increases trust +5. Agents react in character.**

  ## Usage
  ```
  heart [subcommand] [args]
  ```

  ## Description
  Express affection to agent. Increases trust +5. Agents react in character. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `heart` — default view
  - `heart help` — show detailed assistance

  ## Examples
  ```
  heart ada
  ```

  ## Notes
  Affection is a variable they didn't account for.
