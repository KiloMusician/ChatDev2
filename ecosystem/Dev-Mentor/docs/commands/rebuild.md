# REBUILD(1) — Terminal Depths Man Page

## NAME
  rebuild — reconstruct a corrupted or deleted system component

## SYNOPSIS
  rebuild <target> [--from <backup>]

## DESCRIPTION
Rebuild repairs or reconstructs filesystem components, agent memory,
or game state components after corruption, hostile attack, or deliberate
deletion. Some narrative arc paths require rebuild to continue.

After the mole arc, if disinfo caused collateral damage, rebuild restores
affected agent trust gradually. After ascend, rebuild can reconstruct
CHIMERA components under Ghost's control (alternate ending path).

## TARGETS
  rebuild fs           — repair VFS inconsistencies
  rebuild agent <name> — restore damaged agent profile
  rebuild chimera      — reconstruct CHIMERA as ethical system (post-ascend)
  rebuild trust <name> — restore trust after mole incident collateral
  rebuild network      — re-map network after hostile node takeover

## EXAMPLES
  rebuild fs
  rebuild agent cypher
  rebuild chimera
  rebuild trust ada

## REQUIREMENTS
  Some rebuilds require specific keyfiles or story beats.
  `rebuild chimera` requires `arc_discovery_complete` beat.
