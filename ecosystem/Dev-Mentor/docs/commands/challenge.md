# CHALLENGE(1) — Terminal Depths Man Page

## NAME
  challenge — list, accept, or check CTF-style challenges

## SYNOPSIS
  challenge [list|<id>|check <id>]

## DESCRIPTION
Challenges are structured learning missions covering real security skills.
8 categories: web, crypto, pwn, forensics, reverse, osint, networking,
steganography. Each challenge has a difficulty (1–5) and a flag to capture.

Completing challenges grants XP, unlocks agents, and advances narrative arcs.

## SUBCOMMANDS
  challenge list         — show all available challenges
  challenge <id>         — view challenge description
  challenge check <id>   — submit your solution for validation

## CATEGORIES
  web           — web application vulnerabilities
  crypto        — cryptography puzzles and cipher analysis
  pwn           — binary exploitation
  forensics     — file analysis, memory dumps, packet captures
  reverse       — reverse engineering and decompilation
  osint         — open-source intelligence gathering
  networking    — network protocol analysis
  steganography — hidden data in images, audio, documents
