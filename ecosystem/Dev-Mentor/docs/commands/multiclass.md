# multiclass

Secondary class at level 25+.

## Usage
`multiclass [list|choose <class>|status]`

## Description
Gain a secondary class and its associated bonuses after reaching level 25.
Each secondary class grants:
- Access to class-specific command variants.
- +15% XP in that class's skill set.

## Secondary Classes
- `ghost`: Shadow-ghost bonuses
- `hacker`: Shadow-hacker bonuses
- `analyst`: Shadow-analyst bonuses
- `diplomat`: Shadow-diplomat bonuses
- `engineer`: Shadow-engineer bonuses

## Commands
- `multiclass list`: Available secondary classes.
- `multiclass choose <class>`: Select your secondary class.
- `multiclass status`: View your current secondary class and bonuses.

## Rewards
- +15% XP in the secondary class skill set.
