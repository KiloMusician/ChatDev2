# TIMER(1) — Terminal Depths Man Page

## NAME
  timer — display the 72-hour containment countdown

## SYNOPSIS

## DESCRIPTION
Shows the 72-hour containment timer: remaining time, elapsed time, current
loop number, and threshold events approaching. When the timer hits zero, the
loop resets (prestige mechanism). The Temporal Anchor can freeze it.
