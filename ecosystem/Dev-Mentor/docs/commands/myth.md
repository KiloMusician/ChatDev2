# MYTH(1) — Terminal Depths Man Page

## NAME
  myth — query the agent mythology database

## SYNOPSIS

## DESCRIPTION
The Myth database collects oral history and coded mythology surrounding the
59 known operatives in the resistance network. Each agent has a mythological
codename, an origin archetype, and a symbolic role in the ongoing conflict.

With no arguments, lists all known myths. With a codename, displays the full
mythology entry including origin, symbolism, known operations, and ARG hints.
