# MAP(1) — Terminal Depths Man Page

## NAME
  map — display the network node graph

## SYNOPSIS
  map [<node>]

## DESCRIPTION
Renders the Terminal Depths network topology. Nodes represent servers,
facilities, and virtual spaces. Your current position is highlighted.
Connections between nodes show traversal paths (requires hack success or
explicit permission). Some nodes are hidden until discovered via recon.

## NODE CATEGORIES
  residential    — /home/ghost and agent home nodes
  corporate      — NexusCorp and CHIMERA infrastructure
  underground    — Resistance cells and darknet relays
  academic       — Research databases and exploit archives
  government     — Regulatory nodes (unlocked late-game)
  darknet        — Collector, black market, ARG relay nodes

## COMMANDS
  map            — full network graph (ASCII art)
  map <node>     — node detail: connections, owner, difficulty, files

## NAVIGATION
  warp <node>    — instant travel to a discovered node (requires access)
  ssh <node>     — establish session via SSH
  nc <node> <port>— raw netcat connection

  map chimera-control
