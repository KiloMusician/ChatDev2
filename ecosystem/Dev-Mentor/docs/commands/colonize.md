# COLONIZE

  **Node colonization for passive resource generation. Tied to idle_engine.py.**

  ## Usage
  ```
  colonize [subcommand] [args]
  ```

  ## Description
  Node colonization for passive resource generation. Tied to idle_engine.py. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `colonize` — default view
  - `colonize help` — show detailed assistance

  ## Examples
  ```
  colonize status
  ```

  ## Notes
  Every claimed node is a heartbeat in the network.
