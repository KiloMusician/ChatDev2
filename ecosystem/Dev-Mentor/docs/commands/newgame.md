# NEWGAME(1) — Terminal Depths Man Page

## NAME
  newgame — start a fresh game session

## SYNOPSIS
  newgame [<archetype>]

## DESCRIPTION
Creates a new game session, resetting XP, inventory, and story progress.
Previous sessions are preserved and loadable. With an archetype name, starts
the new game with that archetype's loadout.

## EXAMPLES
  newgame
  newgame ghost
  newgame architect
