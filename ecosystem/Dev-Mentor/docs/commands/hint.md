# HINT(1) — Terminal Depths Man Page

## NAME
  hint — request a hint for the active challenge

## SYNOPSIS
  hint [<challenge-id>]

## DESCRIPTION
Requests a hint for the currently active challenge or a specific challenge
by ID. Each hint costs credits. Three tiers of hints are available (light,
medium, heavy) with increasing credit cost and detail.
