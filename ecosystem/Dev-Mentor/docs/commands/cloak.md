# CLOAK(1) — Terminal Depths Man Page

## NAME
  cloak — engage stealth mode

## SYNOPSIS
  cloak [on|off|status]

## DESCRIPTION
Activates stealth mode, reducing your digital footprint. While cloaked:
hack and exploit operations have reduced detection risk, agent monitoring
is harder, and some story events have different outcomes. Cloaking costs
credits over time.

## SUBCOMMANDS
  on      Activate cloak (requires stealth augment)
  off     Deactivate cloak
  status  Current cloak state and credit drain rate

## EXAMPLES
  cloak on

  cloak off
