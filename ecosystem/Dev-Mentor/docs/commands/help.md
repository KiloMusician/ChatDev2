# HELP(1) — Terminal Depths Man Page

## NAME
  help — list available commands or get help for a specific command

## SYNOPSIS
  help [<command>]

## DESCRIPTION
Without arguments, displays all available commands grouped by category with
brief descriptions. With a command name, shows the same output as man <cmd>.
Tab completion works on command names.

## EXAMPLES
  help
  help exploit
  help git
  help challenge

## SEE ALSO
  man, commands, status
