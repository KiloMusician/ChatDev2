# CHAT(1) — Terminal Depths Man Page

## NAME
  chat — open the multi-agent chat console

## SYNOPSIS
  chat [<room>|<agent>]

## DESCRIPTION
Opens the multi-agent chat interface. Without arguments, shows the general
channel. With a room name (hive, council, resistance), opens that channel.
With an agent name, opens a private DM channel with that agent.
