# EXFIL(1) — Terminal Depths Man Page

## NAME
  exfil — trigger data exfiltration from a compromised system

## SYNOPSIS
  exfil [target] [--key <keyfile>]

## DESCRIPTION
Exfiltrates data from a target system after successful exploitation. The CHIMERA
master key is the primary exfiltration target. Secondary targets include agent
dossiers, transmission logs, and the COLLECTOR holding archive.

Requires: active session on target (hack success), valid keyfile.

## TARGETS
  chimera          — extract CHIMERA master key (main objective)
  rescue           — trigger rescue extraction during THE RESCUE arc
  collector        — download Collector auction data
  <nodepath>       — custom target by filesystem path

## EXAMPLES
  exfil chimera
  exfil rescue
  exfil /opt/collector/holding.dat

## KEY FILES
  /opt/chimera/keys/master.key        — required for chimera exfil
  /opt/chimera/keys/secondary.key     — unlocks the WATCHER archive layer
