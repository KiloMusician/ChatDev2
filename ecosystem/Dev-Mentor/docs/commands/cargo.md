# CARGO(1) — Terminal Depths Man Page

## NAME
  cargo — manage in-game cargo and resource logistics

## SYNOPSIS
  cargo [list|ship <item> <dest>|receive|status]

## DESCRIPTION
Manages cargo shipments between game nodes. Ship resources between colony,
darknet, and bazaar nodes. Cargo takes time to arrive and can be intercepted
by hostile agents.

## EXAMPLES
  cargo list
  cargo ship credits colony
  cargo status
