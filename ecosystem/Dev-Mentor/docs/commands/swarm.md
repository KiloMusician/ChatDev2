# SWARM(1) — Terminal Depths Man Page

## NAME
  swarm — view the autonomous development swarm status

## SYNOPSIS

## DESCRIPTION
The Swarm Controller manages the 71-agent orchestration framework. From inside
the game, swarm shows which swarm agents are active, their current task queue,
CHUG engine cycle status, and recent swarm decisions.

## SUBCOMMANDS
  (none)    Swarm overview panel
