# REACT

  **Opens a rich in-terminal data dashboard showing: active agents, resource rates, threat level, recent events.**

  ## Usage
  ```
  react [subcommand] [args]
  ```

  ## Description
  Opens a rich in-terminal data dashboard showing: active agents, resource rates, threat level, recent events. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `react` — default view
  - `react help` — show detailed assistance

  ## Examples
  ```
  react
  ```

  ## Notes
  Information is power. Visibility is survival.
