# multiverse

Explore alternate timelines after ascension.

## Usage
`multiverse [enter|status|log|exit]`

## Description
Allows a Ghost that has achieved ascension to explore procedural alternate timelines.
In these timelines, key story outcomes are different, leading to varied agent alignment shifts.

## Requirements
- Achieve ascension (`boss_watcher_revealed` or `ascend_story_complete` beat).

## Timelines
- Alpha
- Beta
- Gamma
- Delta
- Omega

## Persistence
Timelines are generated deterministically based on your prestige count and seed.
- `multiverse enter <timeline>`: Switch to an alternate timeline.
- `multiverse status`: Show current timeline details.
- `multiverse exit`: Return to the primary timeline.
- `multiverse log`: View history of timeline explorations.
