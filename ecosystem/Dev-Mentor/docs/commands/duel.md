# DUEL(1) — Terminal Depths Man Page

## NAME
  duel — initiate a skill-based contest against an NPC agent

## SYNOPSIS
  duel <agent> [--type hack|social|crypto|recon]

## DESCRIPTION
Challenge an unlocked NPC agent to a skill duel. Duels are mini-games that
pit your skill level against the agent's specialty. Winning increases respect
and earns CRED. Losing reduces fear (if they respect you) or trust (if close).

Duels use the Trust Matrix — agents you outrank in their specialty will
accept; those who outrank you may decline.

## DUEL TYPES
  hack     — head-to-head exploitation (security vs security)
  social   — social engineering match (social skill vs agent charisma)
  crypto   — cipher challenge (ai skill vs agent cryptography rating)
  recon    — network enumeration speed run (networking skill check)

## EXAMPLES
  duel cypher hack
  duel ada social
  duel the_troll crypto

## REWARDS
  Win:  CRED +50–200, respect +5, XP (category) +20–80
  Lose: CRED -10, fear +3 (if agent outclassed you visibly)
