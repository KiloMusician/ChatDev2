# DELEGATE(1) — Terminal Depths Man Page

## NAME
  delegate — delegate a task to an agent

## SYNOPSIS
  delegate <agent> <task>

## DESCRIPTION
Assigns a task to a specific agent for autonomous execution. The agent will
work on the task and report back. Useful for parallelising operations and
for missions that require specific agent skills.

## EXAMPLES
  delegate ada "research chimera's latest exploit"
  delegate raven "recon nexus-core"
  delegate zero "analyze the anomaly log"
