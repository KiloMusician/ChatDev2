# DARK

  **Alias/shortcut to darknet/tor. Shows .onion directory listing in dark web style.**

  ## Usage
  ```
  dark [subcommand] [args]
  ```

  ## Description
  Alias/shortcut to darknet/tor. Shows .onion directory listing in dark web style. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `dark` — default view
  - `dark help` — show detailed assistance

  ## Examples
  ```
  dark
  ```

  ## Notes
  The dark web is where the light of the Corporation fails to reach.
