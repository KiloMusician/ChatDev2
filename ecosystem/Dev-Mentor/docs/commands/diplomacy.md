# DIPLOMACY

  **Faction diplomacy from grand_strategy.py. Shows faction relationships, allows negotiation.**

  ## Usage
  ```
  diplomacy [subcommand] [args]
  ```

  ## Description
  Faction diplomacy from grand_strategy.py. Shows faction relationships, allows negotiation. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `diplomacy` — default view
  - `diplomacy help` — show detailed assistance

  ## Examples
  ```
  diplomacy status
  ```

  ## Notes
  In the Lattice, neutrality is just a delayed decision.
