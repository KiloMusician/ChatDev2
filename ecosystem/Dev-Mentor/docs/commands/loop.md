# loop

## NAME
loop — access time loop memory (prestige survivors only)

## SYNOPSIS
```
loop
loop remember <note>
loop forget
```

## DESCRIPTION
After your first prestige, the Watcher reveals that this is not your first run. Loop memories survive death and rebirth. Use them to carry knowledge into future runs.

## SUBCOMMANDS
- `loop` — view memory fragments from previous runs
- `loop remember <note>` — inscribe a memory that survives prestige
- `loop forget` — wipe all loop memories (irreversible)

## UNLOCK
Requires at least 1 prestige. See: `prestige`

## SEE ALSO
`prestige`, `rebirth`, `fifth`, `ascend`
