# TIMELINE(1) — Terminal Depths Man Page

## NAME
  timeline — view the narrative event timeline

## SYNOPSIS
  timeline [<filter>]

## DESCRIPTION
Renders a chronological list of all significant events in the current
playthrough: story beats triggered, challenges solved, agent interactions,
ARG discoveries, and faction events. Filter by type or agent name.
