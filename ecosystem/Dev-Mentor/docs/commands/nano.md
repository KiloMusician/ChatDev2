# NANO(1) — Terminal Depths Man Page

## NAME
  nano — in-game text editor (nano emulator)

## SYNOPSIS
  nano <file>

## DESCRIPTION
Opens a virtual filesystem file in a nano-compatible editor. Simpler than vi.
Ctrl+X exits, Ctrl+O saves.

## EXAMPLES
  nano /home/ghost/notes.txt
  nano /tmp/payload.sh
