# FORENSICS(1) — Terminal Depths Man Page

## NAME
  forensics — digital forensics and artifact analysis toolkit

## SYNOPSIS
  forensics <subcommand> [file]

## DESCRIPTION
  A suite of tools for analyzing suspicious files, recovering deleted data, and identifying hidden artifacts within the VFS.

## SUBCOMMANDS
  strings    — extract printable strings from binary files
  binwalk    — search for embedded files and executable code
  hexdump    — view file content in hexadecimal format
  steg       — attempt to detect and extract steganographic data
  metadata   — display file system and application metadata

## EXAMPLES
  forensics strings /tmp/.transfer_847b.partial
  forensics hexdump /var/log/.anomaly_0x4A2B.log
  forensics metadata /opt/chimera/cache/.internal_847.memo
