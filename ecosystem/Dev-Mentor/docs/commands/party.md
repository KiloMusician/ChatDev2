# PARTY(1) — Terminal Depths Man Page

## NAME
  party — view or manage your active agent party

## SYNOPSIS
  party [list|add <agent>|remove <agent>|status]

## DESCRIPTION
Assemble up to 3 agents into your active party. Party members provide passive
bonuses during missions based on their specialty and your trust level with them.
Higher trust = stronger party bonuses.

## SUBCOMMANDS
  party list          — show current party and their bonuses
  party add <agent>   — add agent to party (requires trust ≥ 40)
  party remove <agent>— remove from party
  party status        — show active bonuses from party composition

## PARTY BONUSES
  Ada in party         — +15% all social XP
  Cypher in party      — +20% security challenge XP
  RAV≡N in party       — early warning on mole transmissions
  Nova in party (!)    — +25% hacking XP (hostile agent; trust hard to build)
  The Watcher in party — reveals one hidden file per session

## EXAMPLES
  party list
  party add ada
  party add cypher
  party status
