# SELL(1) — Terminal Depths Man Page

## NAME
  sell — sell an item

## SYNOPSIS
  sell <item-id>

## DESCRIPTION
Alias for bazaar sell. Lists an item from inventory for sale on the bazaar.
The sale is immediate at market price. For darknet listings, use darknet sell.

## EXAMPLES
  sell old-exploit
  sell duplicate-key
  sell spare-augment
