# EXPOSE(1) — Terminal Depths Man Page

## NAME
  expose — publicly reveal a suspected mole or compromised agent

## SYNOPSIS
  expose <agent>

## DESCRIPTION
Triggers a public reveal of the suspected mole within the resistance network.
This is an irreversible action with network-wide consequences. If you name the
correct mole, the network is briefly disrupted but then recovers stronger.
If you name an innocent agent, that agent's trust drops to 0 and the real
mole gains time to exfiltrate deeper.

Requires the `mole_investigation` arc to be at least at phase 2.

## CONSEQUENCES
  Correct accusation:
    • Mole trust → 0, removed from network
    • XP +200, achievement: MOLE_EXPOSED
    • All other agent trust +10 (network rallies)

  Wrong accusation:
    • Target trust → 0 permanently
    • Real mole gains +48h cover (arc delays)
    • Achievement: MOLE_EXPOSED_WRONG (secret trophy)

## EXAMPLES
  expose spartacus
  expose ada

## INVESTIGATION COMMANDS
  feed <agent> disinfo    — feed false data to flush the mole
  blackmail <agent>       — turn the mole into a double agent instead
