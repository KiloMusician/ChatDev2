# VI(1) — Terminal Depths Man Page

## NAME
  vi — in-game text editor (vi emulator)

## SYNOPSIS
  vi <file>

## DESCRIPTION
Opens a virtual filesystem file in a vi-compatible editor. Supports basic
vi commands: i (insert), :w (write), :q (quit), :wq (save and quit),
/pattern (search), dd (delete line).

## EXAMPLES
  vi /home/ghost/.diary
  vi /tmp/notes.txt
  vi /opt/chimera/config.yaml
