# ANCHOR(1) — Terminal Depths Man Page

## NAME
  anchor — Temporal Anchor — freeze the 72-hour containment timer

## SYNOPSIS
  anchor [activate|release|status]

## DESCRIPTION
The Temporal Anchor is an advanced artifact that halts the 72-hour containment
loop. When activated the timer stops counting down, giving you unlimited time
to complete objectives. Anchors are consumable — use them wisely.

## SUBCOMMANDS
  activate   Burn one anchor charge to freeze the timer
  release    Unfreeze the timer and resume the countdown
  status     Show anchor charges remaining and current freeze state
