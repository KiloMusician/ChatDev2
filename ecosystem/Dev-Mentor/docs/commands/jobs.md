# JOBS(1) — Terminal Depths Man Page

## NAME
  jobs — list background jobs in the virtual shell

## SYNOPSIS
  jobs

## DESCRIPTION
Lists all background processes spawned in the current session: running
scripts, backgrounded commands, and long-running operations. Background
jobs continue running while you issue other commands.

## EXAMPLES
  jobs
