# PHISHING(1) — Terminal Depths Man Page

## NAME
  phishing — social engineering and credential harvesting

## SYNOPSIS
  phishing [OPTIONS] <target>

## DESCRIPTION
  Deploy targeted social engineering attacks to harvest credentials or install backdoors. Success depends on the quality of OSINT gathered and the target's susceptibility.

## OPTIONS
  --template <type>   use a specific email/message template
  --payload <file>    attach a malicious payload
  --spoof <identity>  impersonate a specific entity

## TARGETS
  - NexusCorp Employees
  - Faction Members (Resistance/Guild)
  - Autonomous Agents

## SUCCESS FACTORS
  - OSINT Depth: More info increases trust.
  - Template Relevance: Match the template to the target's role.
  - Reputation: High faction reputation can make impersonation more effective.

## EXAMPLES
  phishing --template corporate_memo --spoof HR nexus_admin
  phishing --payload reverse_shell.exe nova
