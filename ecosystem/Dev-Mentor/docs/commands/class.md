## NAME
  class — view or set player class

## SYNOPSIS

## DESCRIPTION
Player class defines your specialisation within an archetype. Classes unlock
class-specific commands, dialogue options, and skill multipliers. Without
arguments, shows your current class and available upgrade paths.
