# SKILLS(1) — Terminal Depths Man Page

## NAME
  skills — display your current skill levels and XP

## SYNOPSIS
  skills [<category>]

## DESCRIPTION
Displays the complete skill breakdown for your Ghost operative profile.
Skills are earned by executing real Unix/networking/security commands.
Higher skills unlock harder targets, more powerful commands, and narrative
progression gates.

## SKILL CATEGORIES
  vscode       — VS Code proficiency (earned by debugging, editing)
  git          — Version control skills (commit, diff, merge)
  networking   — Network analysis (nmap, ss, netstat, dig, curl)
  security     — Offensive/defensive ops (hack, exploit, crack, exfil)
  process      — System internals (ps, top, kill, proc traversal)
  tools        — General Unix proficiency (grep, awk, sed, find, etc.)
  social       — Agent relationship building (talk, trust, faction)
  ai           — AI/LLM command usage (ai, ask, ns.llm)
