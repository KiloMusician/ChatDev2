# MORE(1) — Terminal Depths Man Page

## NAME
  more — paginated file viewer

## SYNOPSIS
  more <file>

## DESCRIPTION
Alias for less. Opens a file in a paginated viewer. Space to advance one
screen, q to quit.

## EXAMPLES
  more /var/log/system.log
  more /home/ghost/.diary
