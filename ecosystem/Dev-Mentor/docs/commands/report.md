# REPORT(1) — Terminal Depths Man Page

## NAME
  report — generate a report or call gordon report

## SYNOPSIS
  report [gordon|status|intel|bug <description>]

## DESCRIPTION
Multi-purpose report command. Without arguments or with gordon, calls Gordon's
status report function. status generates a game state report. intel generates
a tactical intelligence briefing. bug reports an in-game bug.
