# CONTEXT(1) — Terminal Depths Man Page

## NAME
  context — show current game context and active systems

## SYNOPSIS
  context [<system>]

## DESCRIPTION
Displays the current operational context: active story arc, faction standing,
equipped augments, active quests, and session metadata. With a system name,
shows context for that specific subsystem.
