# SPY(1) — Terminal Depths Man Page

## NAME
  spy — passive surveillance of an agent or node

## SYNOPSIS
  spy <target> [duration]

## DESCRIPTION
Establishes passive surveillance on a target. Over time, spy collects
intelligence: communication patterns, location data, and behavioural
anomalies. Longer durations yield richer data but increase detection risk.
