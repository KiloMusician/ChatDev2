# EVOLVE(1) — Terminal Depths Man Page

## NAME
  evolve — trigger an agent or system evolution

## SYNOPSIS
  evolve [<agent>|system]

## DESCRIPTION
Triggers an evolution event for an agent or the system. Agent evolution
upgrades their capabilities and changes their personality. System evolution
advances the simulation to the next developmental stage.

  evolve ada
  evolve system
