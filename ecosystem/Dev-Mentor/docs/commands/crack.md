# CRACK(1) — Terminal Depths Man Page

## NAME
  crack — attempt to brute-force or dictionary-attack a target credential

## SYNOPSIS
  crack <target> [--wordlist <list>] [--method brute|dict|hybrid]

## DESCRIPTION
Launches a credential cracking attack against a locked file, service, or
encrypted archive. Success depends on your `ai` and `security` skill levels
vs the target's encryption strength.

Some files are uncrackable without the correct keyfile — see exfil for
key-based decryption. Crack is for credential and password targets.

## METHODS
  brute     — pure brute force. Slow but guaranteed for short passwords.
  dict      — dictionary attack. Fast but limited to wordlist.
  hybrid    — combines dict with rule mutations. Best general approach.

## EXAMPLES
  crack /home/ghost/mission.enc
  crack nexus-db --method dict
  crack collector-archive --wordlist /usr/share/wordlists/rockyou.txt

## WORDLISTS
  /usr/share/wordlists/rockyou.txt    — standard wordlist
  /home/ghost/scripts/custom.lst      — custom list from loot
  /var/data/nexus.wordlist            — NexusCorp credential patterns
