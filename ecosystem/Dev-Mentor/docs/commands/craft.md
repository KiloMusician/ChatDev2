# CRAFT(1) — Terminal Depths Man Page

## NAME
  craft — R4 Crafting System — build items from components

## SYNOPSIS
  craft [bench|list|components|<recipe>|use <item-id>]

## DESCRIPTION
The R4 Crafting System combines salvaged components into usable items: tools,
augment parts, scripts, and artefacts. Check the bench for available recipes
and your components inventory before crafting.

## SUBCOMMANDS
  bench           Show available crafting stations
  list            List all known recipes
  components      Show your component inventory
  <recipe>        Craft the named recipe (e.g. craft cipher-blade)
  use <item-id>   Activate or deploy a crafted item

## EXAMPLES
  craft bench
  craft list
  craft components
  craft cipher-blade
  craft use cipher-blade
