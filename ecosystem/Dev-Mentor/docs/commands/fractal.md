# fractal

ASCII art math dungeons.

## Usage
`fractal [mandelbrot|julia|burning-ship|sierpinski]`

## Description
Explores procedural mathematical spaces using ASCII rendering.
Each type represents a different fractal algorithm:
- `mandelbrot`: The classic complex plane fractal.
- `julia`: Parametric variant (c=-0.7+0.27i).
- `burning-ship`: |Re(z)|+i|Im(z)| variant.
- `sierpinski`: Recursive triangle (8 levels).

## Rewards
- 10 XP (math)
- Achievement: `FRACTAL_MASTER` (after exploring all 4 types)
- Story Beat: `fractal_explorer` (first time)
