# ARCS(1) — Terminal Depths Man Page

## NAME
  arcs — display narrative arc status and progress

## SYNOPSIS
  arcs [<arc_id>]

## DESCRIPTION
Shows all active narrative arcs and their completion status. Narrative arcs
are multi-step story chains that drive the main Terminal Depths plot. Each
arc has 3–5 steps, branching conditions, and XP rewards at each phase.

## ACTIVE ARCS
  betrayal    — The Betrayal (Ada's secondary contact; trust investigation)
  rescue      — The Rescue (Cypher abducted by The Collector)
  discovery   — The Discovery (Ghost's true identity as CHIMERA v0 fragment)
  mole        — The Mole (traitor in the resistance network)

## SUBCOMMANDS
  arcs              — list all arcs with status and phase
  arcs betrayal     — detailed view of betrayal arc
  arcs rescue       — detailed view of rescue arc
  arcs discovery    — detailed view of discovery arc
  arcs mole         — detailed view of mole arc (see also: mole)

## STATUS FLAGS
  inactive    — not yet triggered
  active      — in progress (current step shown)
  completed   — all steps finished
  failed      — failed (branch taken that ended the arc early)
