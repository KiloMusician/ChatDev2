# THEME(1) — Terminal Depths Man Page

## NAME
  theme — change the terminal colour theme

## SYNOPSIS
  theme [<name>|list]

## DESCRIPTION
Changes the Terminal Depths UI colour theme. Available themes: default
(cyberpunk dark), matrix (green on black), blood (red on dark), snow
(light mode), and chimera (purple/gold).

## SUBCOMMANDS
  list     Show all available themes
  <name>   Apply a theme

  theme chimera
