# puzzle

Simulated global community puzzle.

## Usage
`puzzle [community|hint|submit <answer>]`

## Description
Attempt the weekly community puzzle.
- `community`: View the current week's challenge.
- `hint`: Get a hint for 20 XP.
- `submit <answer>`: Submit your solution.

## Puzzle Types
- Cipher (ROT13/Caesar)
- Logic riddle
- Hex/Base64 decode
- Pattern recognition

## Rewards
- 200 XP (on success)
- Achievement: `PUZZLE_SOLVER`
