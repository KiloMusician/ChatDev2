# fifthwall

Opt-in meta-awareness.

## Usage
`fifthwall`

## Description
Breaks the fourth wall and incorporates real-world metadata into the simulation.
Reveals the real system time and day of week as understood by the Watcher.

## Commands
- `fifthwall`: Activate meta-awareness.

## Rewards
- Contextual lore about your world.
- Access to `/proc/realtime` in the VFS.
