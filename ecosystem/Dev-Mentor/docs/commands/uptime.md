# UPTIME(1) — Terminal Depths Man Page

## NAME
  uptime — show system and session uptime

## SYNOPSIS
  uptime

## DESCRIPTION
Displays how long the Terminal Depths server has been running, the current
session duration, number of commands executed this session, and load average
(drawn from the real host for atmosphere).

## EXAMPLES
  uptime

## SEE ALSO
  status, ps, top
