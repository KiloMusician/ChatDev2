# population

## NAME
population — manage your colony population and agent roster

## SYNOPSIS
```
population
population recruit <agent>
population assign <agent> <role>
```

## DESCRIPTION
Your colony grows as you build trust with agents. Each 5 points of total trust across all agents unlocks a new recruit slot. Roles affect resource production rates.

## ROLES
- `hacker` — +10% compute production
- `analyst` — +10% data production
- `medic` — reduces trace accumulation by 5%
- `engineer` — +10% credits production
- `diplomat` — +5 trust with all agents per cycle
