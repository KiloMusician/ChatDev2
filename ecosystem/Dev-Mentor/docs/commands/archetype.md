# ARCHETYPE(1) — Terminal Depths Man Page

## NAME
  archetype — view available player archetypes

## SYNOPSIS
  archetype [<name>]

## DESCRIPTION
Player archetypes define your starting skill loadout and narrative alignment.
Without arguments, lists all archetypes. With a name, shows full details:
starting skills, XP bonuses, locked commands, and lore background.
