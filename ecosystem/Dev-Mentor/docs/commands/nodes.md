# NODES(1) — Terminal Depths Man Page

## NAME
  nodes — display the network node map

## SYNOPSIS
  nodes [<filter>]

## DESCRIPTION
Renders the in-game network topology: known hosts, their status (online/
offline/compromised), open ports, and connection paths. Filter by faction
or status to focus the view.
