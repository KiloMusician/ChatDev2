# ID(1) — Terminal Depths Man Page

## NAME
  id — print real and effective user/group IDs

## SYNOPSIS
  id

## DESCRIPTION
Shows the current virtual user identity: uid, gid, groups, privilege level,
and clearance tier. Root context and special groups (agents, chimera, lattice)
unlock additional commands and filesystem access.
