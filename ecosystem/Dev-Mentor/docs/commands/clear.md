# CLEAR(1) — Terminal Depths Man Page

## NAME
  clear — clear the terminal screen

## SYNOPSIS
  clear

## DESCRIPTION
Clears all output from the current terminal buffer. The prompt and cursor
remain. Useful for reducing visual noise during long sessions. Ctrl+L also
triggers clear on most keyboard configurations.

## EXAMPLES
  clear

  help, status
