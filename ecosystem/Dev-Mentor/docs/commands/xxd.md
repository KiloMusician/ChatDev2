# XXD(1) — Terminal Depths Man Page

## NAME
  xxd — hex dump of a file

## SYNOPSIS
  xxd <file>

## DESCRIPTION
Creates a hexadecimal dump of a virtual filesystem file, showing both the
hex values and ASCII representation. Essential for CTF reverse engineering
and forensics challenges.

## EXAMPLES
  xxd /tmp/payload.bin
  xxd /opt/chimera/core/.internal_847.memo
