# PRESTIGE(1) — Terminal Depths Man Page

## NAME
  prestige — enter prestige mode after ascension

## SYNOPSIS
  prestige [confirm]

## DESCRIPTION
After completing the main story arc (ascend), prestige resets your character
level to 1 but preserves: augments, agent trust scores, arc memory, and a
permanent +10% XP multiplier per prestige cycle.

Each prestige cycle introduces new content: harder node variants, deeper ARG
layers, additional Watcher transmissions, and alternate story branch unlocks.

Prestige requires: level 30+, main arc completed, `first_ascend` beat present.

## PRESTIGE BONUSES (per cycle)
  +10% global XP multiplier
  +5 flat to all skill check starting values
  1 new augment slot (up to 3 extra)
  Access to prestige-exclusive myth entries

## PRESTIGE COST
  All XP reset to 0
  Level reset to 1
  Active challenge progress cleared
  (Trust, arcs, augments, and lore preserved)
