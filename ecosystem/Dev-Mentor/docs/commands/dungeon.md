# DUNGEON

  **Dungeon crawler via dungeon_engine.py. Mathematical dungeon rooms.**

  ## Usage
  ```
  dungeon [subcommand] [args]
  ```

  ## Description
  Dungeon crawler via dungeon_engine.py. Mathematical dungeon rooms. This command allows the operator to interact with specialized subsystems of the Terminal Depths environment.

  ## Subcommands / Options
  - `dungeon` — default view
  - `dungeon help` — show detailed assistance

  ## Examples
  ```
  dungeon start
  ```

  ## Notes
  The Basement holds what the Library refuses to archive.
