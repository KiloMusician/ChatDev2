# INVENTORY(1) — Terminal Depths Man Page

## NAME
  inventory — display your current items, keys, and loot

## SYNOPSIS
  inventory [--filter keys|loot|augments|cred]

## DESCRIPTION
Shows all items Ghost has acquired: cryptographic keys, looted files,
active augments, and current CRED balance. Items persist across sessions.

## ITEM CATEGORIES
  Keys         — cryptographic credentials (required for exfil/crack)
  Loot         — extracted files (evidence, intel, exploit code)
  Augments     — active cybernetic enhancements (see: augment)
  CRED         — in-game currency (used for augment purchases, bribes)

## EXAMPLES
  inventory
  inventory --filter keys
  inventory --filter loot
  inventory --filter cred

## KEY ITEMS

  secondary.key           — WATCHER archive key (ARG layer)
