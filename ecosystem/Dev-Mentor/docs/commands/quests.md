# QUESTS(1) — Terminal Depths Man Page

## NAME
  quests — display active quests or generate new procedural missions

## SYNOPSIS
  quests [generate]

## DESCRIPTION
Lists your active quest log: main story quests, faction quests, and
procedurally generated missions. generate uses the QuestGenerator to create
new contextual missions based on your current arc, level, and agent states.
Three procgen quests are auto-seeded on each new session.

## SUBCOMMANDS
  (none)    Show current quest log
  generate  Generate up to 3 new procgen quests

  quests generate
