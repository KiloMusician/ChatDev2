# SS(1) — Terminal Depths Man Page

## NAME
  ss — another utility to display virtual socket statistics

## SYNOPSIS
  ss [-t|-u|-l|-n|-p]

## DESCRIPTION
Displays socket statistics similar to netstat. ss is faster and provides
more detail about TCP/UDP connections, socket states, and process bindings
in the virtual network simulation.

## EXAMPLES
  ss -tln
  ss -p
  ss
