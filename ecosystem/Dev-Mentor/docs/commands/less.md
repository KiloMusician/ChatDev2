# LESS(1) — Terminal Depths Man Page

## NAME
  less — view file with pagination

## SYNOPSIS
  less <file>

## DESCRIPTION
Opens a virtual filesystem file in a paginated viewer. Navigate with arrow
keys or j/k. q to quit. / to search. G to jump to end. g to jump to start.

## EXAMPLES
  less /home/ghost/.diary
  less /var/log/system.log
