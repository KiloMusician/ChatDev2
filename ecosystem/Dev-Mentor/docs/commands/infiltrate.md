# INFILTRATE(1) — Terminal Depths Man Page

## NAME
  infiltrate — infiltrate a target organisation or node

## SYNOPSIS
  infiltrate <target> [--deep|--surface|--stealth]

## DESCRIPTION
Long-term infiltration operation against a target organisation. Unlike a
quick hack, infiltration embeds a presence inside the target network/org
over time, providing persistent intelligence and multiple access points.

## OPTIONS
  --deep     Deep cover infiltration (slow but thorough)
  --surface  Surface-level access only (faster, less risk)
  --stealth  Prioritise not being detected

## EXAMPLES
  infiltrate nexuscorp --deep
  infiltrate chimera --stealth
  infiltrate resistance --surface

  hack, backdoor, social, recon
