# MEDITATE(1) — Terminal Depths Man Page

## NAME
  meditate — pause and gain passive XP

## SYNOPSIS
  meditate [<seconds>]

## DESCRIPTION
Enter a meditative state for passive XP generation. Each meditate cycle
awards 1-5 XP in a random skill tree based on recent activity. Also
restores any depleted stamina for hack/battle operations.

## EXAMPLES
  meditate
  meditate 30
