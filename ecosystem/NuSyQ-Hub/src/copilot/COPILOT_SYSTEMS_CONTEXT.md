# 🤖 Copilot Systems Directory Context

## 📋 Directory Purpose
**Primary Function**: GitHub Copilot enhancements, workspace optimization, context management, and AI-assisted development tools for the KILO-FOOLISH ecosystem.

## ✅ Files That BELONG Here

- **Workspace Enhancers**: Copilot workspace optimization tools (`copilot_workspace_enhancer.py`)
- **Context Managers**: Copilot context generation and management systems
- **Enhancement Bridges**: Copilot enhancement and augmentation utilities
- **AI Assistants**: Copilot-powered development assistance tools
- **Code Analyzers**: Copilot-enhanced code analysis and suggestions
- **Context Propagators**: Systems that share context with Copilot
- **Development Accelerators**: Copilot-powered development workflow tools

## ❌ Files That Do NOT Belong Here

- **General AI Systems**: Non-Copilot AI belongs in `src/ai/`
- **Core System Logic**: Core functionality belongs in `src/core/`
- **Testing Files**: Belong in `tests/`
- **Integration Bridges**: General integration belongs in `src/integration/`
- **User Interfaces**: UI components belong in `src/interface/`

## 🔗 Integration Points

- **Core Systems**: Enhances core development workflow
- **AI Systems**: Coordinates with general AI infrastructure
- **Development Tools**: Integrates with `src/tools/` for enhanced development
- **Context Systems**: Shares context across the entire ecosystem

## 🏷️ Required Tags

All files must include OmniTag/MegaTag headers with:
- **#COPILOT_SYSTEM** - Primary classification
- **#WORKSPACE_ENHANCEMENT** - If enhancing workspace functionality
- **#CONTEXT_MANAGEMENT** - If managing Copilot context
- **#DEVELOPMENT_ASSISTANT** - If providing development assistance
