# Chronicles of the Eternal Bastion

**Defend. Evolve. Ascend.**

---

## 🎮 About

An incremental tower defense RPG that combines:
- **Idle/Incremental mechanics** - Progress even while offline!
- **Tower Defense** - Strategic placement and upgrades
- **RPG Elements** - Character progression, skills, equipment
- **Deep Story** - 3-act narrative with multiple endings
- **NPC System** - Dialogue, quests, relationships
- **Meta-Progression** - Ascension system for endless replayability

---

## 🚀 How to Play

```bash
python chronicles_eternal_bastion.py
```

### Requirements

```bash
pip install pygame
```

### Controls

- **Mouse**: Place towers, select UI elements
- **1-6**: Quick-select tower types
- **Space**: Start/Pause waves
- **Q/W/E/R**: Use character abilities
- **Tab**: Open character sheet
- **I**: Inventory
- **M**: Map
- **ESC**: Menu/Pause

---

## 🎯 Features

### Incremental System
- **Passive Income**: Earn gold and essence continuously
- **Offline Progress**: Keep earning while away
- **Automation**: Unlock auto-upgrade and auto-deploy
- **Ascension**: Prestige for permanent bonuses

### Tower Defense
Six tower types:
- **Archer's Perch** - Long range physical damage
- **Mage Tower** - AoE magical damage
- **Barracks** - Spawns soldiers to block enemies
- **Artillery** - Extreme range, slow fire rate
- **Temple** - Support buffs for nearby towers
- **Summoner's Circle** - Adaptive elemental spawns

Enemy types scale with difficulty!

### RPG Progression
- **5 Classes**: Warrior, Mage, Ranger, Cleric, Necromancer
- **Stats**: STR, INT, DEX, WIS, CHA
- **Equipment**: 8 slots with 6 rarity tiers
- **Skills**: Active abilities with cooldowns
- **Talent Trees**: 3 branches per class

### Story & NPCs
- **Commander Aldric**: Quest giver and story narrator
- **Sage Elara**: Knowledge keeper and magic shop
- **Blacksmith Gorin**: Equipment upgrades
- **Mysterious Stranger**: Ascension guide
- **Merchant Kira**: General shop

**Three Acts** with branching narrative and multiple endings!

## 🎲 Game Loop

1. **Hub** - Talk to NPCs, upgrade gear, prepare
2. **Battle** - Place towers, fight waves, earn rewards
3. **Progress** - Level up character, unlock new abilities
4. **Ascend** - Prestige for permanent bonuses
5. **Repeat** - With increased power and new challenges!

## 🔮 The Story

You are the last Commander of the Eternal Bastion, a fortress that has stood for millennia against the forces of the Void. Each night, corruption grows stronger, sending waves of twisted creatures.

But you possess a secret: the power to **Ascend**, retaining knowledge across lifetimes. Each defense teaches you more about the ancient war that sundered your world.

**Will you break the cycle? Or become part of it?**

## 🎯 Quick Start

1. **Install** - `pip install pygame`
2. **Run** - `python chronicles_eternal_bastion.py`
3. **Play** - Start at the menu, enter the Hub, then Battle!
4. **Defend** - Place towers (costs 100 gold each)
5. **Earn** - Passive income + wave rewards
6. **Progress** - Level up and unlock new powers

## 🏆 Tips

- **Early game**: Focus on Archer towers for cost-effectiveness
- **Mid game**: Mix tower types for balanced defense
- **Late game**: Specialize with masteries and talent trees
- **Meta**: Ascend early and often for compounding bonuses
- **NPCs**: Build relationships for better shop prices
- **Story**: Explore all dialogue options for lore
