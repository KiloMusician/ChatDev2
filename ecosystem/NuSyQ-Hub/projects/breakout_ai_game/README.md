# Breakout Game with Paddle and Bricks

A classic arcade-style breakout game built using HTML, CSS, and JavaScript.

## Game Description
Breakout is a simple yet addictive game where the player controls a paddle at the bottom of the screen to bounce a ball up and break bricks along the top. The objective is to destroy all the bricks by hitting them with the ball. Collecting power-ups can enhance gameplay, such as increasing paddle size or enabling extra lives.

## How to Install
1. Download the ZIP file from the repository.
2. Extract the contents to your local machine.
3. Open the `index.html` file in your web browser.

## How to Run
Simply open the `index.html` file in any modern web browser. The game will load automatically, and you can start playing immediately.

## Controls
- **Left Arrow Key**: Move the paddle left.
- **Right Arrow Key**: Move the paddle right.
- **Spacebar**: Start the game or reset after losing a life.

## Features
- **Dynamic Bricks**: Different colors indicate varying levels of hardness. Harder bricks require more hits to break.
- **Power-Ups**: Randomly spawned power-ups can provide temporary boosts, such as increased paddle size or extra lives.
- **Scoring System**: Points are awarded for breaking each brick and collecting power-ups.
- **Level Progression**: Increase difficulty by adding new levels with progressively smaller bricks.

Enjoy the game and have fun!
