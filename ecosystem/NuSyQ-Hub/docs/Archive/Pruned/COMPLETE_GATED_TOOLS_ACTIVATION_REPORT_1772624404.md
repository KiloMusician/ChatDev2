## 💡 Best Practices

### 1. Prefer Specialized Tools
```python
# ✅ GOOD: Use Pylance for Python
pylanceRunCodeSnippet("print('Hello')")

# ❌ AVOID: Terminal with escaping issues
run_in_terminal('python -c "print(\'Hello\')"')
```

### 2. Validate Before Operating
```python
# Mermaid: Validate → Preview
mermaid-diagram-validator(code)
mermaid-diagram-preview(code)

# Python: Check syntax → Execute
pylanceSyntaxErrors(code)
pylanceRunCodeSnippet(code)
```

### 3. Use Async Agents for Heavy Tasks
```python
# Delegate to Copilot coding agent
github_create_pull_request_with_copilot(
    owner, repo,
    problem_statement="Complex refactoring task",
    title="Refactor module X"
)
```

### 4. Batch Operations
```python
# Push multiple files at once
github_push_files(owner, repo, branch, files, message)

# Not: github_create_or_update_file() x 10
```

### 5. Activate On-Demand
```python
# Only activate when needed
if need_security_scan:
    activate_github_tools_security_management()
    github_list_dependabot_alerts(owner, repo)
```
