# Evolve API

Endpoints:

- GET `/api/evolve` — List existing evolve suggestion artifacts persisted under `Reports/rosetta`.
- POST `/api/evolve` — Trigger an evolve suggestion run.
  - Request body (JSON):
    - `prompt` (string, optional): seed text for the suggestion.
  - Response: `{ "file": "<path>", "content": { ... } }`

Artifacts:

- Suggestions are written to `Reports/rosetta/evolve_suggestion_<timestamp>.json`.

Notes:

- The POST handler prefers to call `src.tools.rosetta_runner.run_suggest(prompt)` when available; otherwise it writes a lightweight suggestion artifact.
- The endpoint accepts a JSON body and is documented in the OpenAPI schema generated by FastAPI.
