## 🎬 How to Use

### Basic Usage

```python
from src.tools.agent_task_router import AgentTaskRouter

router = AgentTaskRouter()

# Analyze with Ollama
result = await router.route_task(
    task_type="analyze",
    description="Check code quality",
    target_system="ollama",
)

# Generate with ChatDev
result = await router.route_task(
    task_type="generate",
    description="Create REST API",
    target_system="chatdev",
    context={"project_name": "MyAPI"},
)

# Auto-route intelligently
result = await router.route_task(
    task_type="review",
    description="Review architecture",
    target_system="auto",  # Let orchestrator decide
)
```

### Run Demonstrations

```bash
# All demos
python demo_agent_hub.py

# Specific demo
python demo_agent_hub.py ollama
python demo_agent_hub.py chatdev
python demo_agent_hub.py consciousness
```

### Run Tests

```bash
# All tests
pytest tests/test_agent_task_router.py -v

# Specific test class
pytest tests/test_agent_task_router.py::TestChatDevRouting -v

# With coverage
pytest tests/test_agent_task_router.py --cov=src.tools.agent_task_router
```

### Quick Validation

```bash
python validate_hub.py
```
