## 🚀 Advanced Usage

### Full Autonomous Mode (Use with Caution)
```powershell
# System runs completely autonomously - NO human approval
.\scripts\start_autonomous_systems.ps1 -Mode full -Interval 30
```

**⚠️ WARNING:** Only use after extensive testing in supervised mode.

### Custom Interval
```powershell
# Run every 15 minutes (more aggressive)
.\scripts\start_autonomous_systems.ps1 -Interval 15

# Run every 2 hours (conservative)
.\scripts\start_autonomous_systems.ps1 -Interval 120
```

### Integration with Systemd (Linux)
```bash
# Create systemd service
sudo nano /etc/systemd/system/nusyq-autonomous.service

[Unit]
Description=NuSyQ-Hub Autonomous Development Loop
After=network.target

[Service]
Type=simple
User=youruser
WorkingDirectory=/path/to/NuSyQ-Hub
ExecStart=/usr/bin/python3 -m src.automation.autonomous_loop --mode supervised --interval 30
Restart=on-failure
RestartSec=60

[Install]
WantedBy=multi-user.target

# Enable and start
sudo systemctl enable nusyq-autonomous
sudo systemctl start nusyq-autonomous
sudo systemctl status nusyq-autonomous
```

### Integration with Windows Task Scheduler
```powershell
# Create scheduled task to run at startup
$action = New-ScheduledTaskAction -Execute "python" -Argument "-m src.automation.autonomous_loop --mode supervised --interval 30" -WorkingDirectory "C:\path\to\NuSyQ-Hub"
$trigger = New-ScheduledTaskTrigger -AtStartup
$principal = New-ScheduledTaskPrincipal -UserId "$env:USERNAME" -RunLevel Highest
Register-ScheduledTask -TaskName "NuSyQ Autonomous Loop" -Action $action -Trigger $trigger -Principal $principal
```
