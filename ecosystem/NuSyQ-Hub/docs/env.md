# Environment Variables for NuSyQ-Hub

Document all required and optional environment variables here. Example:

- `OPENAI_API_KEY` (required): Your OpenAI API key for LLM integration
- `OLLAMA_API_KEY` (optional): Ollama API key for local LLMs
- `CHATDEV_PATH` (optional): Path to ChatDev installation
- `DATABASE_URL` (optional): Database connection string
- `SECRET_KEY` (required): Secret key for cryptographic operations

Add more as needed for your deployment.
