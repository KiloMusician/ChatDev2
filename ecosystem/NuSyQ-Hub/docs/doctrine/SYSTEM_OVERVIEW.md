## AI Agent Roles (In This System)

### Claude — The Architect
**Cost:** High per token, deep reasoning  
**Best For:** Compression, abstraction, long-horizon planning, decision records  
**Output:** Artifacts that save future tokens  
**Avoid:** Mechanical edits, repeated scans, boilerplate

### Copilot — The Operator
**Cost:** Low, always available  
**Best For:** Precise file edits, incremental changes, following explicit plans  
**Output:** Clean, reversible commits with diffs

**Cost:** Essentially free (37.5GB local models)

## Core Principles

1. **Durable artifacts > conversation**  
   If reasoned once, capture it permanently.

2. **Memory > repetition**  
   Never rediscover what was already paid for.

3. **Small, reversible > large, brittle**  
   Progress in stable notches.

4. **Evidence > assumptions**  
   Diffs before commits, tests before push.

5. **Calm > speed**  
   Slow clarity creates permanence.
