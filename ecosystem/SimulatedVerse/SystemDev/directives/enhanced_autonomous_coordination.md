# Enhanced Autonomous Coordination Patterns
## Based on RLCI v1 Battle-Tested Experience

### **INFRASTRUCTURE-FIRST SUPREMACY**

**What We Learned**: Surface fixes are productivity killers. Always investigate root architectural issues first.

**New Directive**: Before any feature work, always:
1. **Capability Audit**: Run `tsx SystemDev/scripts/capability_registry.ts` to know your tools
2. **Beacon Check**: Read relevant `.beacon.json` to understand safe zones
3. **Context Load**: Generate RLCI envelope to carry rich context forward
4. **Evidence Setup**: Prepare receipt generation before starting work

**Why This Works**: Prevents tool conflicts, reduces mobile debugging pain, enables systematic progress vs random thrashing.

### **QUANTUM-WINK METHODOLOGY VALIDATION**

**Micro-Cycle Discipline** (≤8 edits, ≤400 lines):
- Maintains focus and reviewability
- Enables rapid iteration without architectural debt
- Prevents scope creep and analysis paralysis
- Creates clear evidence trail for AI learning

**Receipt Discipline** (every cycle documented):
- Enables momentum detection and cascade triggers
- Provides learning feedback for AI improvement
- Creates accountability without micromanagement
- Allows sophisticated coordination without theater

**Anti-Theater Enforcement**:
- Only real, measurable progress gets tracked
- No psychological satisfaction without delivery
- Evidence-based vs claim-based progress
- Systematic debugging vs random fixes

### **MOBILE-FIRST DEVELOPMENT REALITY**

**Samsung S23 As Primary Target**: Most constraints come from mobile; desktop is easy.

**New Mobile-First Rules**:
1. **Cache Defusing**: Always set `Cache-Control: no-store` for development
2. **Service Worker Blocking**: Implement `unstick.json` kill switches
3. **WebGL1 Fallback**: Never assume WebGL2; mobile compatibility requires v1
4. **Threading Disabled**: Mobile WebView doesn't handle SharedArrayBuffer reliably
5. **Touch-First UI**: Large buttons, mobile debugging without DevTools required

**Mobile Debugging Stack**:
- Preview HUD with `?hud=1` toggle for on-screen logs
- Council Bus integration for live event streaming
- Console interception since DevTools unavailable
- Touch-optimized control overlays
