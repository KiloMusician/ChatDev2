# KPulse Knowledge Base

This is the versioned Obsidian vault for the KPulse ecosystem. All design decisions, specifications, and insights are documented here and automatically indexed by RepoPilot for RAG queries.

## Structure

- `specs/` - Technical specifications and system architecture
- `decisions/` - Architecture decision records (ADRs)  
- `playbooks/` - Operational guides and procedures
- `insights/` - Analytics insights and player behavior patterns
- `research/` - Research notes and exploration logs

## Auto-Generation

This knowledge base is automatically updated through:
- Nightly log analysis → insights summaries
- RepoPilot suggestions → decision records
- Analytics reports → balance recommendations
- Player feedback → improvement proposals

## Usage

RepoPilot indexes this entire vault to answer questions like:
- "Why did we choose the 16-step gradient system?"
- "What are the current balance issues with Tier 8?"
- "How should we handle colonist mood mechanics?"

All content here becomes part of the institutional memory and helps guide future development decisions.
