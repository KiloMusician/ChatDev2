## 🚀 Getting Started

### **Starting the Garden**

```python
from TempleOfKnowledge.Floor06_RooftopGarden.autonomous_custodians import (
    RooftopGardenOrchestrator, CustodianPersonality
)

# Initialize garden
library = KnowledgeLibrary()
garden = RooftopGardenOrchestrator(library)

# Create diverse custodians
garden.create_custodian("Sage", CustodianPersonality.CONTEMPLATIVE)
garden.create_custodian("Spark", CustodianPersonality.ENERGETIC)
garden.create_custodian("Harmony", CustodianPersonality.HARMONIOUS)

# Start peaceful cultivation
await garden.start_garden_activities()
```
