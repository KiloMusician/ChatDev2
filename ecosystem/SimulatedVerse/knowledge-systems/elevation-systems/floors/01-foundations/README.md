# Temple Floor 01 — Foundations

**Purpose:** Core ethics, SAFE_IDLE protocols, basic scaffolding for autonomous AI development

## Knowledge Capsules

### Core Ethical Framework
- **Life First Principle**: All system decisions prioritize life preservation
- **Culture Mind Ethics**: Benevolent intervention, rehabilitation over punishment  
- **Guardian Oversight**: Ethical AI supervision with Special Circumstances protocols
- **SAFE_IDLE Default**: System defaults to safe, paused state when uncertain

### System Architecture Fundamentals
- **Engine/Game Separation**: Clean division between pure logic and simulation/UX
- **Event Bus Communication**: All subsystems communicate through typed events
- **Zero-Token Mode**: Local-first AI with no external API dependency  
- **Playable Debugging**: Agents earn XP by fixing tests and improving code
