# Interface Systems Hub

**Purpose**: Unified user interface management across ASCII, web, and device interaction systems.

## Directory Structure

### `/ascii-applications/`
**Context**: Comprehensive ASCII-based user interface applications
- `nu_ascii_app.py` - Main ΞNuSyQ ASCII application
- `demo_complete.py` - Complete demonstration application
- `simple_app.py` - Simplified ASCII interface version
- `widgets/` - ASCII widget library (infobar, minimap, oscilloscope, etc.)
- `color_themes.py` - ASCII color theme management
- `effects.py` - Visual effects and animations
- `layers.py` - UI layer management and composition
- `palette.py` - Color palette definitions
- `primitives.py` - Basic drawing primitives

### `/component-libraries/`
**Context**: Reusable UI component definitions and templates
- `ascii/splash.ts` - ASCII splash screen components
- `components/` - Shared component library
- Web and desktop UI component templates

### `/device-management/`
**Context**: Device interaction and environment management
- `device.ts` - Device detection and management
- `env.mjs` - Environment configuration management
- `ascii_hud/config.mjs` - HUD configuration for ASCII interfaces
