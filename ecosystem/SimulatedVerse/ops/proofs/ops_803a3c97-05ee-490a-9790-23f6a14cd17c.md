# Ops Task: Collect UI console errors

**Collecting UI Console Errors**

We'll use the `ndjson` library to read the ndjson file and log any errors found.

```typescript
import * as fs from 'fs';
const ndjson = require('ndjson');

// Read from ndjson file
const reportStream = fs.createReadStream('./reports/ui_console.ndjson');
const parser = ndjson.parse(reportStream);

// Collect errors
let errorCount = 0;
let lastError: any;

parser.on('data', (obj) => {
  if (obj.error) {
    errorCount++;
    lastError = obj.error;
    console.error(`Error ${errorCount}: ${lastError.message}`);
  }
});

parser.on('end', () => {
  console.log(`Total errors collected: ${errorCount}`);
});
```

This script reads the `ui_console.ndjson` file, parses each object in it, and logs any error messages found. Finally, it outputs a total count of collected errors.

Make sure to install the `ndjson` library first using npm: `npm install ndjson`.
