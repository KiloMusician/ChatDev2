# Ops Task: Generate uptime log

**Solution**

To generate an uptime log, we'll use the `nx` CLI command to create a JSON file containing the uptime data.

```bash
# Create the reports directory if it doesn't exist
nx mkdirp 'reports'

# Generate uptime log in JSON format and save it to reports/uptime_log.json
nx exec --shell "echo '{\"timestamp\": $(date +%s), \"uptime\": $(awk '/^bogomips/{print $1; exit} 0 /proc/cpuinfo | sort -n)}' > reports/uptime_log.json"
```

**Explanation**

* We first create the `reports` directory using `nx mkdirp`.
* Then, we use `nx exec --shell` to execute a shell command. This command uses `awk` and `sort` to extract the bogomips value from `/proc/cpuinfo`, which is used to calculate uptime.
* The uptime data is then printed in JSON format with the current timestamp and saved to `reports/uptime_log.json`.

**Note**: Make sure you have the `nx` CLI installed and properly configured.
