# Ops Task: Wire mt.011 agent productivity metrics to reports/agent_productivity.json

Here is the solution in TypeScript:

```typescript
import axios from 'axios';
import { fs } from '@xfdb/core';

const mtAgentProductivityMetrics = await axios.get('https://example.com/mt/011/productivity/metrics')
    .then(response => response.data);

fs.writeFileSync('./reports/agent_productivity.json', JSON.stringify(mtAgentProductivityMetrics, null, 2));
```

This script fetches the agent productivity metrics from a remote API and writes them to a file named `agent_productivity.json` in the `./reports` directory.
